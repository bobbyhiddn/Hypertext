# Hypertext Style Guide

## Visual Identity

### Card Dimensions
- Orientation: Vertical (portrait)
- Aspect ratio: 2:3
- Target resolution: 1024 x 1536 pixels (digital)
- Print resolution: 2048 x 3072 pixels minimum (300 DPI at 2.5 x 3.5 inches)

### Color Palette

| Role | Color | Usage |
|------|-------|-------|
| Primary | Navy (#1a2744) | Borders, filled pips, text |
| Secondary | Gold (#c9a227) | Trim, accents, outlines |
| Background | Warm Parchment (#f4e8d1) | Card background |
| Ink | Near-black (#1a1a1a) | Body text |

### Typography

| Element | Style |
|---------|-------|
| Title (Word) | Classic serif display, large, navy |
| Gloss | Italic serif, smaller, below title |
| Labels | Uppercase small caps, gold or navy |
| Body text | Clean serif, high legibility |
| Greek | Must render correctly with proper diacritics |
| Hebrew | Must render correctly, right-to-left |

## Card Frame

### Border Structure
- Outer border: Thick navy
- Inner trim: Thin gold
- Corners: Rounded with subtle filigree
- Background: Antique parchment with subtle grain texture

### Layout Sections (top to bottom)

1. **Header Row**
   - Left: Collector ID (#001)
   - Left (below): Type badge (NOUN, VERB, etc.)
   - Center: Word title + gloss
   - Right: Rarity icon + text

2. **Art Panel**
   - Height: ~44% of card
   - Frame: Gold inner frame with navy outline
   - Content: Single coherent scene, no text, no modern objects

3. **Stats Row**
   - Three stats: LORE, CONTEXT, COMPLEXITY
   - Each: 1-5 circular pips
   - Filled: Navy with gold outline
   - Empty: Parchment with navy outline

4. **Content Panels**
   - ABILITY: One short ability line
   - OT VERSE: Reference + snippet
   - NT VERSE: Reference + snippet
   - GREEK/HEBREW split panel
   - TRIVIA: 3-5 bullet points

## Iconography

### Rarity Pill (Top Right)
The rarity indicator appears in the top-right corner as a navy pill containing:
1. Rarity text (e.g., "COMMON", "RARE")
2. A diamond icon to the right of the text

| Rarity | Pill Background | Text Color | Diamond Fill |
|--------|-----------------|------------|--------------|
| COMMON | Navy (#1a2744) | White | White |
| UNCOMMON | Navy (#1a2744) | White | Green (#4a7c59) |
| RARE | Navy (#1a2744) | White | Gold (#c9a227) |
| GLORIOUS | Navy (#1a2744) | White | Orange (#f28c28) |

**Important**: The pill background is always navy. Only the diamond fill color varies by rarity. The diamond shape is consistent across all rarities (4-sided rhombus).

### Stat Pips
- Shape: Circle only (no other shapes)
- Filled: Solid navy fill with gold outline
- Empty: Parchment fill with navy outline
- Maximum: 5 pips per stat

## Render Rules

### Must Include
- All defined sections rendered once
- Proper language rendering (Greek, Hebrew RTL)
- Exact rarity icon shape and text
- Circular stat pips only

### Must Avoid
- Text inside artwork
- Extra borders outside card
- Top microtext (no "GAME: HYPERTEXT")
- Duplicate section headings
- Watermarks or captions outside card
- Gradients on icons
- Rendering hidden metadata fields
