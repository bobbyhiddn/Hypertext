# Type Icons Rubric
# Reference: templates/palettes/type_symbols_palette.png

TYPE ICON SPECIFICATIONS:

1. NOUN - Closed Book
   - Shape: Rectangular book shape with visible spine on left
   - Features: Slightly curved corner on right, single line suggesting page edges
   - Style: Simple, slightly rounded edges

2. VERB - Slanted Pencil
   - Shape: Long, slanted cylinder tapering to a point
   - Features: Rounded eraser at top, viewed from side angle
   - Style: Clear pencil shape with distinct eraser end

3. ADJECTIVE - Sparkle Pencil
   - Shape: Pencil with decorative elements
   - Features: Two small four-pointed stars emanating from middle
   - Style: Stars represent the descriptive/shiny quality

4. NAME - Feather Quill
   - Shape: Single feather/quill
   - Features: Classic quill pen shape
   - Style: Elegant writing implement

5. TITLE - Ornate Frame
   - Shape: Empty picture frame
   - Features: Decorative/ornate border design
   - Style: Suggests framing/presentation

ICON PLACEMENT:
- Located inside the Type Circle (navy circle with gold border)
- Icon color: WHITE on navy background
- Icon should be clearly visible and centered within the circle
